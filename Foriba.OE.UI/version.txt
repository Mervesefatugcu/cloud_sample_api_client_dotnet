=================================================================================================================================
 Örnek Foriba Bulut API test projesi oluşturuldu.

 e-Fatura  

 Yapılan İşler:  

     1. e-Fatura mükellef bilgileri sorgulaması 
     2. e-Fatura ve uygulama yanıtlarını web servis kullanarak gönderme
     3. e-Fatura ve uygulama yanıtı zarflarının web servis kullanarak statülerinin sorgulanması
     4. Gelen ve gönderilen zarflı veya zarfsız e-fatura ve uygulama yanıtlarının web servis kullanarak sistemden alınması
     5. Gelen ve gönderilen e-faturaların web servis kullanarak  html, pdf ve xml formatında datalarının alınması

 e-İrsaliye  

 Yapılan İşler:

     1. e-İrsaliye mükellef bilgileri sorgulaması 
     2. e-İrsaliye ve e-irsaliye yanıtlarını web servis kullanarak  gönderme
	 3. e-İrsaliye ve e-irsaliye yanıtı zarflarının web servis kullanarak statülerinin sorgulanması
     4. Gelen ve gönderilen zarflı veya zarfsız e-irsaliye ve e-irsaliye yanıtlarının web servis kullanarak sistemden alınması
     5. Gelen ve gönderilen e-irsaliye ve e-irsaliye yanıtlarının web servis kullanarak html, pdf ve xml formatında datalarının alınması

 e-Arşiv

 Yapılan İşler:

     1. e-Arşiv faturalarının web servis kullanarak gönderilmesi
     2. e-Arşiv faturalarının web servis kullanarak html, pdf ve xml formatında datalarının alınması
